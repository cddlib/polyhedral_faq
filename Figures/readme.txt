This folder contains several eps (encapsulated postscript) files created
for the Polyhedral Computation FAQ.  All graphic images were created
with Mathematica using VertexEnum-notebook and Mathematica packages
(in particular, VertexEnum.m) which are available at 
https://library.wolfram.com/infocenter/MathSource/440/ .
